🚀 Key Features
Dynamic Search: Fetch current weather conditions for any city globally.
Auto-Location: Integrated Geolocation API to detect user coordinates and provide local weather instantly.
Comprehensive Data: Displays temperature, weather descriptions (e.g., "Clear Sky", "Haze"), humidity levels, and wind speeds.
Glassmorphism UI: A sleek, modern aesthetic using backdrop-filter blur effects and semi-transparent layers.
Mobile Optimized: Specifically tuned for high-performance viewing on mobile devices (tested on Android/Redmi).
🛠️ Technical Stack
Frontend: HTML5 (Semantic tags), CSS3 (Flexbox, Grid, Custom Properties).
Scripting: Vanilla JavaScript (Asynchronous Programming, Fetch API).
API: OpenWeatherMap API.
Deployment: GitHub Pages.
